# Поиск контакта

### Client
- `cd client`
- Устанавливаем зависимости `npm i`
- Запускаем локально `npm run start`
- Для создания билда `npm run build`

  ### Backend
  - `cd backend`
  - Устанавливаем зависимости `npm i`
  - Запускаем локально `npm run dev`
  - Компиляция и запуск локально `npm run start`
  - Только компиляция `npm run build`
